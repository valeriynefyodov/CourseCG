User's Manual:
The program draws a polylines between the points selected by user.

Left click - set the point of the polyline
Right click - stop drawing the polyline

Alt+F - File
Alt+E - Edit
Alt+H - Help

Alt+F, N - New file
Alt+F, S - Save file
Alt+F, O - Open file
Alt+F, X - Exit
Alt+E, D - Delete the last drawn line
Alt+H, A - About
